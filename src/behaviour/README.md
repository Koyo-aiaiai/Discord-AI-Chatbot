# Behvaiour

This is responsible for staggering the replies of the LLM such that it feels more natural and human.

## Things it should be able to do

- [ ] Turn control: Understanding when it is the AI's turn to respond.
- [ ] Random Reply Time: LLM not just replying instantly every time. Timing responses based on urgency and also randomness.
- [ ] Urgency Control: Understanding when the AI should respond right away and when it doesn't need to. (Also includes whether the AI is currently in conversation and thus should respond quickly).
